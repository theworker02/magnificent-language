MGL

Magnificent Language

A readable systems-flavored language starter kit with a real lexer, parser, interpreter, CLI, and docs.

Read the README Explore syntax

What ships in the box

  • Custom MGL syntax with braces and no semicolon requirement
  • Lexer, AST parser, tree-walking interpreter, and interactive REPL
  • CLI commands for help, version, program execution, and project scaffolding
  • Class support, methods, constructors, functions, loops, arrays, and imports
  • Built-in standard library for output, input, file I/O, type inspection, and conversion

Sample program

class Car {
  func init(name) {
    self.name = name
  }

  func drive() {
    print(self.name + " is driving")
  }
}

let car = Car("Nova")
car.drive()
